Библиотека Panini

Простой генератор файлов для Gulp. Он компилирует HTML-страницы, используя общий макет. Эти страницы также могут включать части HTML, хелперы, такие как Handlebars или внешние данные в формате JSON или YAML.

Панини не является полноценным шаблонизатором. Он решает очень специфическую задачу сборки HTML файлов из общих элементов, используя язык шаблонов.

Настраиваем в Gulp

			.pipe(panini({
				root: `./src/pages`,
				layouts: `./src/layouts`,
				data: `./src/data`,
				partials: `./src/partials`,
				helpers: `./src/helpers`
			}))

Используем

			---
			subject: Make text in your browser larger with ZoomSupport
			description: Make text in your browser larger with ZoomSupport
			---
			  {{> pre-header }} // Подключаем блок из папки pages
			  {{> sub-header }}
			  {{> content key=value}} // Можем передать переменны в блок
			  {{> footer }}

Тестирование

  1. Возможность отправки письма непосредственно через SMTP
  2. Отправка на тестовые серсисы (emailonacid)
  3. Интеграция с Litmus

Тестирование через SMTP

				"mail": {
			    "to": ["youtestemail@email.com"],
			    "from": "Company name info@company.com",
			    "smtp": {
			      "auth": {
			        "user": "youtestemail@email.com",
			        "pass": "**********"
			      },
			      "host": "smtp.gmail.com",
			      "secureConnection": true,
			      "port": 465
			    }
			  }

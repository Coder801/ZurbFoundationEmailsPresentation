Установка

			  //Установка глобально Command line interface (CLI)
			  npm install --global foundation-cli
		
			  //Инициализация проекта
			  foundation new --framework emails
		
			  //Переходим в папку с проектом и запускаем
			  npm start

Файловая структура

			assets/* Файлы стилей и картинки.
			layouts/* Обертка тела вашего письма.
			pages/* HTML страницы писем.
			partials/* Реиспользуемые компоненты
			data/* Данные письма (тексты, переводы).
